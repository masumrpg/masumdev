# API Reference

## QRCode Component

:::tip Component Configuration
The `QRCode` component provides a highly customizable interface for generating QR codes with various styles and configurations.
:::

### Core Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| value | string | - | Required. The content to be encoded in the QR code |
| variant | QRCodeVariant | 'BASIC' | Design variant of the QR code |
| size | number | 256 | Size of the QR code in pixels |
| version | QRCodeVersion | - | QR code version (1-40) |
| errorCorrectionLevel | 'L' \| 'M' \| 'Q' \| 'H' | 'M' | Error correction capability |

### Style Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| color | string | '#000' | Main color of QR code elements |
| backgroundColor | string | 'transparent' | Background color |
| includeBackground | boolean | false | Add background wrapper |
| gradient | QRCodeGradientConfig | - | Gradient configuration |

### Advanced Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| logo | LogoOptions | - | Logo configuration |
| piece | PieceOptions | - | Individual QR piece styling |
| eye | EyeOptions | - | Corner eye pattern styling |
| maxVersion | QRCodeVersion | - | Maximum QR code version |

## QRCodeScanner Component

:::tip Scanner Configuration
The `QRCodeScanner` component offers a feature-rich interface for QR code scanning with customizable UI and behavior.
:::

### Core Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| onSuccessfulScan | (state: OnSuccessfulScanProps) => void | Yes | Scan success callback |
| validate | QRCodeValidator | No | Custom validation function |

### Scanning Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| cooldownDuration | number | 2000 | Time between scans (ms) |
| scanningArea | ScanningAreaProps | - | Target area configuration |

### UI Controls Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| showControls | boolean | true | Show/hide control buttons |
| showStatus | boolean | true | Show/hide status info |
| showTorchButton | boolean | true | Show/hide torch control |

### Appearance Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| theme | 'light' \| 'dark' | 'light' | UI theme selection |
| overlayStyle | OverlayStyle | - | Scanner overlay styling |
| frameStyle | FrameStyle | - | Scanner frame styling |
| cornerStyle | CornerStyle | - | Corner markers styling |
| statusStyle | StatusStyle | - | Status display styling |

### Callbacks Props

| Prop | Type | Description |
|------|------|-------------|
| onTorchChange | (isOn: boolean) => void | Torch state change handler |
| onScanStart | () => void | Scan start handler |
| onScanEnd | () => void | Scan end handler |
| onCooldownStart | (duration: number) => void | Cooldown start handler |
| onCooldownEnd | () => void | Cooldown end handler |

### Haptics Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| enableHapticFeedback | boolean | true | Enable/disable haptics |
| customHapticFeedback | HapticFeedbackConfig | - | Custom haptics settings |

### Navigation Props

| Prop | Type | Description |
|------|------|-------------|
| onBackPress | () => void | Back button handler |
| showBackButton | boolean | Show/hide back button |
| backButtonIcon | string | Back button icon name |

### Custom Components Props

| Prop | Type | Description |
|------|------|-------------|
| renderCustomStatus | (statusInfo: StatusInfo) => React.ReactNode | Custom status renderer |
| renderCustomControls | (controls: ControlsInfo) => React.ReactNode | Custom controls renderer |
